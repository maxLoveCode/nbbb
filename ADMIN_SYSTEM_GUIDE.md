# NBBB 管理后台使用指南

## 访问地址

管理后台访问地址：`http://你的域名/admin/`

## 主要功能

### 1. 商品管理

**功能描述：**
- 查看所有商品列表
- 搜索商品（支持按关键词、分类搜索）
- 查看商品详情
- 商品来自聚水潭ERP系统，会自动同步

**使用方法：**
1. 点击左侧菜单"产品管理"
2. 使用搜索框输入关键词或分类名称进行筛选
3. 点击"查看"按钮查看商品详细信息
4. 支持分页浏览，每页显示20/50/100条数据

### 2. 订单查看

**功能描述：**
- 查看所有订单列表
- 按订单状态筛选（待支付、已支付、已发货、已完成、已取消）
- 按日期范围筛选
- 查看订单详情（包括商品清单、收货信息等）

**使用方法：**
1. 点击左侧菜单"订单管理"
2. 使用筛选条件查找特定订单
3. 点击"查看"按钮查看订单详情
4. 订单详情包括：
   - 订单基本信息（订单号、状态、创建时间等）
   - 收货信息（收货人、电话、地址）
   - 商品清单（商品名称、规格、数量、价格）

### 3. 图片上传

**功能描述：**
- 上传图片到阿里云OSS
- 支持批量上传（最多10张）
- 支持的格式：jpg, jpeg, png, gif, webp
- 单个文件最大10MB
- 自动生成图片URL，方便复制使用

**使用方法：**
1. 在"商品管理"页面点击"上传图片"按钮
2. 输入上传目录（默认为 `images`，可以自定义如 `products`, `banners` 等）
3. 选择要上传的图片文件（可多选）
4. 点击"上传"按钮
5. 上传完成后，会显示每张图片的URL
6. 点击"复制URL"按钮将图片URL复制到剪贴板

**阿里云OSS配置（通过环境变量）：**
- `OSS_BUCKET`：Bucket名称（默认：nbbb）
- `OSS_ACCESS_KEY_ID`：阿里云 AccessKey ID
- `OSS_ACCESS_KEY_SECRET`：阿里云 AccessKey Secret
- `OSS_REGION`：区域（默认：oss-cn-hangzhou）

### 4. 会员查看

**功能描述：**
- 查看所有注册会员
- 搜索会员（支持按昵称、手机号搜索）
- 查看会员详情（包括订单数、累计消费等）
- 按状态筛选（启用/禁用）

**使用方法：**
1. 点击左侧菜单"用户管理"
2. 使用搜索框输入昵称或手机号进行查找
3. 点击"查看"按钮查看会员详细信息
4. 会员详情包括：
   - 基本信息（ID、昵称、手机号、微信OpenID）
   - 注册时间和最后登录时间
   - 订单统计（订单数量、累计消费）
   - 账户状态

### 5. 首页管理

**功能描述：**
- 管理小程序首页的轮播图（Banners）
- 管理横向轮播（Lower Swiper）
- 管理三图展示（Three Images）

**使用方法：**
1. 点击左侧菜单"首页管理"
2. 在对应的模块中添加、编辑或删除内容
3. 所有内容支持排序和启用/禁用
4. 图片URL可以使用上传功能获得

### 6. 商品描述管理

**功能描述：**
- 为聚水潭商品添加本地描述
- 本地描述会与聚水潭的商品描述拼接显示

**使用方法：**
1. 点击左侧菜单"商品描述管理"
2. 输入商品编码（如：NBB-AWJ003）
3. 点击"加载当前描述"查看现有描述
4. 在文本框中输入或修改本地描述
5. 点击"保存描述"保存更改

## API端点

管理后台使用以下API端点：

- **商品管理：** `/api/admin/products`
- **订单管理：** `/api/admin/orders`
- **用户管理：** `/api/admin/users`
- **图片上传：** `/api/admin/upload`
- **首页管理：** `/api/admin/homepage`
- **商品描述：** `/api/admin/product-descriptions`

## 技术栈

**前端：**
- Bootstrap 5.1.3
- Bootstrap Icons 1.7.2
- 原生JavaScript

**后端：**
- Node.js + Express
- PostgreSQL 数据库
- 阿里云OSS存储
- Multer 文件上传处理

## 注意事项

1. **图片上传：** 
   - 上传的图片会存储在阿里云OSS，请确保OSS配置正确
   - 建议使用合适的目录结构组织图片，如 `products/`, `banners/` 等
   - 图片URL生成后可直接用于商品、首页等模块

2. **数据同步：**
   - 商品数据来自聚水潭ERP，需要定期同步
   - 订单数据实时更新

3. **权限管理：**
   - 目前管理后台未启用认证，建议在生产环境中启用 `adminAuth` 中间件
   - 相关代码在各路由文件中已注释，需要时取消注释即可

4. **性能优化：**
   - 列表页面支持分页，建议合理设置每页显示数量
   - 图片建议使用CDN加速访问

## 故障排除

### 如果管理后台无法访问：

1. 检查后端服务是否运行：
   ```bash
   curl http://localhost:3000/api/admin/health
   ```

2. 检查Nginx配置是否正确（如果使用Nginx）

3. 查看后端日志：
   ```bash
   tail -f /nbbb/backend.out.log
   ```

### 如果图片上传失败：

1. 检查阿里云OSS配置是否正确
2. 确认AccessKey有写入权限
3. 检查文件大小是否超过10MB限制
4. 确认文件格式是否支持（jpg, jpeg, png, gif, webp）

### 如果数据加载失败：

1. 检查浏览器控制台错误信息
2. 检查后端API是否正常响应
3. 确认数据库连接是否正常

## 联系支持

如有问题，请查看：
- 后端日志：`/nbbb/backend.out.log`
- 浏览器控制台（F12开发者工具）
- 后端服务状态：`http://localhost:3000/api/admin/health`
