### 映射规则

| Tab | source | 映射来源 | 商品数 |
|-----|--------|----------|--------|
| 博主甄选 | activity | Excel 导入（运营手选） | 13 |
| 人宠同款 | activity | Excel 导入（运营手选） | 15 |
| 男款上装 | category | 上装类商品 | 20 |

## 三、表结构

### category_page_categories

| 字段 | 说明 |
|------|------|
| id | 主键 |
| name | Tab 名称 |
| source | `activity`=运营专题，`category`=商品分类 |
| sort_order | 排序 |

### category_page_products

| 字段 | 说明 |
|------|------|
| category_id | 关联的 Tab ID |
| product_code | 商品编码 |
| sort_order | 在 Tab 中的排序 |

## 五、API 使用

```bash
# 获取分类页数据
GET /api/category-page

# 获取分类商品（分页）
GET /api/category-page/:id/products?pageNum=1&pageSize=10

# 管理接口
GET  /api/category-page/config
GET  /api/category-page/:id/product-codes
POST /api/category-page/:id/products
DELETE /api/category-page/:id/products/:code
```
