# NBBB Storefront

## Stack
- Next.js App Router
- React 19
- Server Components for SEO pages

## Environment
- `API_BASE_URL`: backend origin
- `NEXT_PUBLIC_SITE_URL`: storefront public origin

## Start
```bash
npm install
npm run dev
```

## Current Scope
- 首页、分类、详情、搜索、品牌页、Lookbook、专题页、购物车、结算、账户页骨架
- `sitemap` 与 `robots`
- 对接 `/api/web/*` 的独立站 BFF
