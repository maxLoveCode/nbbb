# Storefront MVP Roadmap

## Phase 1
- 完成 `storefront/` 的 Next.js 前台工程。
- 交付首页、分类页、详情页、搜索页、购物车页、结算页和基础账户页。
- 打通 `/api/web/home`、`/api/web/categories`、`/api/web/listing`、`/api/web/products/:code`。
- 保留 SSR、`sitemap`、`robots` 和基础页面元信息。

## Phase 2
- 打通 Web 注册登录、收藏、地址和订单查询。
- 补活动专题页、品牌故事页、Lookbook 内容页。
- 增加埋点、首单券提示、免邮门槛与推荐商品模块。

## Phase 3
- 增加优惠券、会员权益、多语言、多币种和营销自动化。
- 为 SEO 增加博客 / 编辑内容体系。
- 将 CMS 扩展到首页、专题页、品牌页和 Lookbook 页。
